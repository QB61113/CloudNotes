# 模糊查询

> like
> between and
> in 
> si null | is not null



## 一、like

### 1.1 注意

- 字符串要用`""`
- 要使用通配符
   - 通配符
      - `%`任意多个字符，包含0个字符（%指在这个字符的前后还有其他字符）
      
      - `_`任意单个字符

### 4.1 注意
