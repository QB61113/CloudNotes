# 基础查询语法特点和细节处理

## 一、基础查询

### 1.1 语法与特点

语法：
> `select 查询列表 from 表名;`

特点：
> 1. 查询列表可以是：表中的字段、常量、表达式、函数。
> 1. 查询的结果是一个虚拟的表格。



### 1.2 查询

查询表中的字段：
> `select 字段 from 表名`

查询表中的多个字段：
> `select 字段，字段，字段 from 表名;`

查询表中的所有字段：
> `select 字段，字段，..... from 表名;`（可单击选择）

> 或    `select * from 表名;`（按顺序查询）
