# 条件运算符的使用

> 1. 查询价格>20的书籍

```mysql
select * from tb_books where book_price>20;
```
![img](https://gitee.com/xleixz/CloudNotes-Images/raw/master/Typora-Images/20220425132412.png)



> 2. 查询书籍编号不等于3的书籍

```mysql
select * from tb_books where ID <>3;
```
